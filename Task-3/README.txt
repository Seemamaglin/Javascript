## Task-3:IMAGE GALLERY LIGHTBOX PROJECT

## Features:
-> Responsive image gallery using CSS Grid
-> Clickable thumbnails using Class toggles
-> Used Modal/lightbox overlay for enlarged image view
-> Smooth fade and zoom transition effects
-> Close button to exit the lightbox using <span> tag
-> Class toggling used to show and hide the modal

How It Works:

1. Thumbnail images are displayed in a grid layout.
2. JavaScript event listeners detect clicks on gallery images.
3. When an image is clicked, the lightbox image source is dynamically updated.
4. A CSS class is added to the lightbox to make it visible.
5. CSS transitions provide smooth visual effects.
6. Clicking the close button removes the active class and hides the lightbox.
