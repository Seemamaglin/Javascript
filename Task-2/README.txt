## Task-2:SIMPLE CALCULATOR 

## FEATURES IMPLEMENTED
    ->Basic arithmetic operations: +, -, *, /
    ->Dynamic display update while typing
    ->Clear all (C) functionality
    ->Clear last character (Backspace)
    ->Error handling for invalid expressions
    ->Single toggle button for brackets ( )
    ->Centered calculator layout using Flexbox
    ->Hover effects for buttons

## CALCULATOR LOGIC OVERVIEW
    ->User input is captured through button clicks
    ->Inputs are appended to the display dynamically
    ->Expression is validated before calculation
    ->Invalid cases like multiple operators together are detected and try-catch is used to safely evaluate expressions
    ->Errors are displayed as "Error" instead of crashing the app
